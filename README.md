## 📌 Overview
A Medical Symptom to Disease Prediction app built using NLP and Machine Learning.
Enter symptoms in plain text and get the top predicted diseases with confidence scores.

## 🛠️ Tech Stack
- Python, Scikit-learn, NLTK-style NLP Pipeline
- TF-IDF Vectorization + Random Forest Classifier
- Streamlit (deployed on Streamlit Cloud)
